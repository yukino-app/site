# Number

## `Number.infinity`

Returns infinity.

```
Number.infinity
# Infinity
```

## `Number.negativeInfinity`

Returns negative infinity.

```
Number.negativeInfinity
# -Infinity
```

## `Number.NaN`

Returns `NaN`.

```
Number.NaN
# NaN
```

## `Number.maxFinite`

Returns the maximum value of number.

```
Number.maxFinite
# 1.7976931348623157e+308
```

## `Number.from`

Takes in a value and returns a number.

```
Number.from("10");
# 10

Number.from(11);
# 10
```

## `Number.fromOrNull`

Takes in a value and returns a number or `null`.

```
Number.from("10");
# 10

Number.from(list []);
# null
```
